# Debug & Internal Editor Plugin for CDM

CDM (Custom Discord-bot Maker) のための強力なデバッグおよびプラグイン開発ツールです。VS Code のようなインターフェースをブラウザ上で提供し、プラグインの内部構造の確認、リアルタイム編集、一括検索・置換などを可能にします。

## 🚀 主な機能

### 1. VS Code ライクなエディタ UI
- **エクスプローラー**: インストール済みのプラグインをフォルダ階層で表示。
- **マルチファイル編集**: `manifest.json`（メタデータ）と `plugin.js`（実行コード）を個別に編集可能。
- **リアルタイム保存**: `Ctrl + S` で変更を即座に保存。

### 2. 高度な検索と置換
- **グローバル検索**: 全プラグインのソースコードから特定の文字列を検索。
- **一括置換**: 複数のプラグインにまたがる文字列をワンクリックで一括置換。

### 3. プラグイン管理
- **新規作成**: エディタ上から新しいプラグインフォルダを直接作成。
- **UUID 確認**: 各プラグインに割り当てられた一意の識別子（UUID）を確認。
- **属性管理**: スタイル干渉、ブロック干渉、自作フラグなどのメタデータを JSON 形式で編集。

### 4. システムログ (OUTPUT)
- 保存、作成、エラーなどのシステムイベントをリアルタイムでコンソールに表示。

## 🛠 インストール方法

1. 本リポジトリまたは提供された `debug-plugin.zip` をダウンロードします。
2. CDM のプラグイン管理画面を開きます。
3. 「ZIPからインストール」ボタンをクリックし、ダウンロードしたファイルを選択します。
4. 画面右下に表示される 「🛠 Debug Console」ボタンをクリックしてツールを起動します。

## ⚠️ 注意事項

- **自作フラグ**: 本プラグインは `isCustom: true` に設定されています。有効化している間、CDM の共有機能はセキュリティ保護のため制限（非表示）されます。
- **反映**: `plugin.js` の変更を反映させるには、保存後に CDM を再読み込みする必要があります。
